# Loading Indicator
Designed by [Oleg Frolov](https://dribbble.com/shots/5421606-Loader-XL) and implemented by me in Python using PySide6

## Dependencies
- [PySide6](https://pypi.org/project/PySide6/)

## Running
> Run the commands below depending on your system
> ## **Windows**:
```console
python main.py
```
> ## **MacOS and Linux**:
```console
python3 main.py
```
